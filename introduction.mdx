Ruger Bot is a Discord bot management platform built for server administrators who want more control without the complexity. Log in with your Discord account, select a server you manage, and instantly access a full-featured dashboard where you can configure commands, moderate members, review activity logs, and grant dashboard access to trusted team members — no separate account required.

## Get Up and Running

Follow these four steps to go from a fresh login to a fully configured server dashboard.

<Steps>
  <Step title="Log in with Discord">
    Visit [rugerbotapp.com](https://rugerbotapp.com) and click **Login with Discord**. Authorize Ruger Bot to read your Discord profile and server list — this is how we identify which servers you manage.
  </Step>
  <Step title="Add Ruger Bot to Your Server">
    From the server selection screen, click **Add to Server** next to any server where you hold the **Manage Server** permission. Complete the Discord OAuth prompt to invite the bot and grant it the permissions it needs to operate.
  </Step>
  <Step title="Open Your Server Dashboard">
    Once the bot joins, your server appears in the Ruger Bot dashboard. Click it to open the server overview, where you'll find real-time activity, member stats, and quick-access controls.
  </Step>
  <Step title="Configure Commands and Access">
    Head to the **Bot Configuration** section to enable or disable commands, set command prefixes, and customize auto-responses. Optionally, link additional Discord accounts to share dashboard access with moderators or co-admins.
  </Step>
</Steps>

<Note>
  Ruger Bot requires the **Manage Server** permission on your Discord account to display a server in your dashboard. Make sure you hold this permission on any server you intend to manage. Additionally, the bot itself needs appropriate role permissions within the server — see [Add Bot to Server](/getting-started/adding-the-bot) for the full permissions list.
</Note>
