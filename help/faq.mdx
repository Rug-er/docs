Ruger Bot makes it easy to manage your Discord bot from a clean web dashboard — no code required. If you're just getting started or running into something unexpected, the questions below cover the most common topics our users ask about.

<Accordion title="What is Ruger Bot?">
  Ruger Bot is a Discord bot management platform. You control your bot entirely from a web dashboard — configure commands, set up auto-moderation, manage welcome messages, and review activity logs without ever touching code. Each server you manage gets its own independent dashboard with a full suite of controls.
</Accordion>

<Accordion title="Is Ruger Bot free?">
  Ruger Bot offers a free tier that covers single-server use with core features included. If you manage multiple servers or need advanced features such as extended log retention and priority support, you may need to upgrade to a premium plan. Check the Billing section of your dashboard for a full breakdown of what each plan includes.
</Accordion>

<Accordion title="What Discord permissions does Ruger Bot need?">
  Ruger Bot requests the following permissions when you add it to your server:

  - **Manage Messages** — to delete or pin messages as part of moderation actions
  - **Kick Members** — to remove members when auto-mod or manual moderation rules are triggered
  - **Ban Members** — to permanently ban members who violate your server rules
  - **Send Messages** — to deliver welcome messages, command responses, and alerts
  - **Embed Links** — to send rich-format embeds for logs and notifications
  - **Read Message History** — to evaluate messages for auto-mod rules
  - **View Channels** — to access and monitor configured channels

  <Note>
    Ruger Bot only uses these permissions in the channels and contexts you configure. You can review and restrict channel-level permissions directly in Discord's server settings.
  </Note>
</Accordion>

<Accordion title="Why don't I see my server in the dashboard?">
  The dashboard only displays servers where your Discord account holds the **Manage Server** permission. If a server is missing, check the following:

  - Confirm you are logged in with the correct Discord account — the one that has Manage Server rights on that server.
  - Verify your role in Discord has the Manage Server permission enabled.
  - Make sure you haven't removed Ruger Bot from that server, since disconnected servers are hidden from the dashboard list.

  Log out, switch accounts if necessary, and log back in to refresh your server list.
</Accordion>

<Accordion title="Can multiple people access the same server dashboard?">
  Yes. You can grant other Discord accounts access to your server's dashboard without sharing your own login credentials. Navigate to **Manage Access** inside the server dashboard, then link the Discord accounts you want to invite. Each person signs in with their own Discord account and sees only the servers they have been granted access to.
</Accordion>

<Accordion title="Is my data stored securely?">
  Ruger Bot stores only the minimum data required to operate:

  - Your **Discord user ID** (used for authentication and access control)
  - Your **server configuration** (commands, permissions, welcome messages, auto-mod rules)
  - **Activity logs** generated by bot actions on your server

  Ruger Bot does **not** store the content of your members' messages. All stored data is encrypted at rest, and you can request deletion of your data at any time through the dashboard settings.
</Accordion>

<Accordion title="How do I remove Ruger Bot from my server?">
  You have two options:

  1. **From the dashboard** — Open your server dashboard, go to **Bot Controls**, and click **Disconnect Bot**. This removes the bot and deactivates all configurations.
  2. **From Discord** — Open your Discord server settings, navigate to **Members** or **Integrations**, find Ruger Bot, and kick or remove it directly.

  <Warning>
    Removing the bot disables all active features immediately. Your settings are preserved for 30 days in case you re-add the bot later.
  </Warning>
</Accordion>

<Accordion title="What happens to my settings if I remove the bot?">
  Your full configuration — including commands, permissions, auto-mod rules, and welcome message templates — is preserved for **30 days** after you remove the bot. If you re-add Ruger Bot to your server within that window, all of your previous settings are automatically restored. After 30 days, the configuration is permanently deleted.
</Accordion>

<Accordion title="Can I export my activity logs?">
  Yes. Open the **Logs & Activity** section of your server dashboard, then click the **Export** button in the top-right corner of the log table. Ruger Bot generates a CSV file containing your filtered log entries and downloads it directly to your browser. You can filter by date range, action type, or moderator before exporting to narrow the data set.
</Accordion>

<Accordion title="How do I report a bug?">
  You have two easy ways to report a bug:

  - Click the **Feedback** button in the footer of any dashboard page and describe what went wrong. The form automatically attaches your session metadata to help the support team investigate.
  - Join the **Ruger Bot support server** via the invite link in the dashboard footer and post in the `#bug-reports` channel.

  <Tip>
    Include a screenshot and the steps that led to the issue — this speeds up the resolution process significantly.
  </Tip>
</Accordion>

<Accordion title="Why is the bot showing as offline?">
  The bot may be in the middle of a scheduled restart or a brief service interruption. Wait **60 seconds**, then check the status indicator in your dashboard header. If the bot remains offline, open **Bot Controls** and click **Force Config Reload** to prompt a fresh connection. If the problem persists after two reload attempts, report it via the Feedback button or the support server.
</Accordion>

<Accordion title="Can I use Ruger Bot on multiple servers?">
  Yes — Ruger Bot supports as many servers as your plan allows. Each server gets its own fully independent dashboard with separate configurations, logs, and access controls. Changes you make to one server's settings have no effect on any other server. Navigate between your servers using the server selector in the top-left corner of the dashboard.
</Accordion>
